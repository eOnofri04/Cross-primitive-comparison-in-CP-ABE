# Cross-primitive-comparison-in-CP-ABE

[![DOI](https://img.shields.io/badge/DOI-10.1515/jmc--2025--0051-blue)](https://doi.org/10.1515/jmc-2025-0051)
[![DOI](https://img.shields.io/badge/DOI-10.69091/koine/vol--8--P10-blue)](https://doi.org/10.69091/koine/vol-8-P10) 

This repository holds the material related to the pubblication  
**"<a href="https://doi.org/">Cross-primitive comparison in CP-ABE: Bilinear Pairing vs. Lattices</a>"**  
by *[Lorenzo Gasparini](0009-0006-4651-3583), [Elia Onofri](https://eliaonofri.it), Martina Palmucci, and [Marco Pedicini](https://orcid.org/0000-0002-9016-074X)*.  
The work was publishe within Volume 20, no. 1 (2026) of Journal of Mathematical Cryptology (**[JMC](https://www.degruyterbrill.com/journal/key/jmc/html)**).

A preliminary version of this work: **"<a href="https://doi.org/10.69091/koine/vol-8-P10">Comparing Lattices and Bilinear Parings in CP-ABE</a>"**  
by *[Lorenzo Gasparini](0009-0006-4651-3583), [Elia Onofri](https://eliaonofri.it), Martina Palmucci, and [Marco Pedicini](https://orcid.org/0000-0002-9016-074X)*  
which was presented within the *Financial Cryptography in Rome Conference* (**[FCiR25](https://www.decifris.it/fcir25)**) held in Rome, Italy, October 1st, 2025,  
and published within the Volume **[FCiR25 ACTA](https://doi.org/10.69091/koine/vol-8-I01)** by *De Cifris Koine*.

---

## Description

This repository contains the implementation carrying out the experiments supporting the results described in the above papers.
